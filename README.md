# graph-theory-network-analysis
Real-world applications of Graph Theory in Software Engineering: Implementing Dijkstra's for routing and PageRank for node importance.
## Project Overview
This project was developed as part of my M.Sc. curriculum at Gauhati University. 
It explores the mathematical foundations of Graph Theory and its practical 
implementation in modern software ecosystems.

### Key Objectives:
* **Path Optimization:** Implementing Dijkstra’s Algorithm to simulate efficient data routing.
* **Network Centrality:** Analyzing node importance within a social graph to identify influential users.
* **Complexity Analysis:** Evaluating the efficiency of adjacency lists vs. matrices in Python.
